# SeaFormer Web 工作记录

## 项目目标

- 在当前 `seaformer` 可板端推理的基础上，新增一个可在局域网访问的中文 Web 服务。
- 支持单张上传、批量上传、`FP/INT8` 模型切换。
- 支持根据分割结果做 `OK/NG` 判定。
- 支持展示纯推理时间和推理阶段内存占用。
- 默认不保存上传结果，但前端可切换为保存完整结果。
- 预留视频流检测入口，但本期不实现。

## 已确认需求

- 项目根目录：`/home/elf/segment`
- 当前实际可写路径：`/userdata/home_elf/segment`
- 新项目目录：`/home/elf/segment/seaformer-web`
- Web 服务端口：`8000`
- 页面语言：中文
- 当前仅接入 `seaformer`
- 批量结果默认最多展示 `3` 项，超出后点击展开
- 判定阈值按前景像素数计，默认 `0`，前端可调整
- 判定逻辑：前景像素数 `>` 阈值则为 `NG`，否则为 `OK`
- 支持返回批量中 `OK` 的文件名列表
- 保存结果开启时保存完整输出

## 技术路线

- 后端：`FastAPI`
- 前端：服务端模板 + 原生 JavaScript
- 推理：直接复用 `rknn/seaformer/output/hybrid` 下的整模型 `.rknn`
- 板端运行时：`rknn-toolkit-lite2 2.3.2`

## 目录规划

- `app/main.py`
  - FastAPI 入口
- `app/inference.py`
  - SeaFormer 模型加载、推理与结果落盘
- `app/templates/index.html`
  - 中文 Web 页面
- `app/static/styles.css`
  - 页面样式
- `app/static/app.js`
  - 前端逻辑
- `runtime/tmp`
  - 临时结果目录
- `runtime/saved`
  - 用户启用保存时的结果目录
